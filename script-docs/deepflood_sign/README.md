# DeepFlood 自动签到脚本

## 📋 脚本简介

这是一个用于 **DeepFlood 论坛（deepflood.com）** 的自动签到脚本。  
支持 **固定签到（默认）** 和 **随机签到** 模式切换。

---

## ⚙️ 配置说明

### 1. 环境变量（必需）

| 变量名 | 说明 | 示例 |
|------|------|------|
| DEEPFLOOD_COOKIE | 用户的 Cookie 字符串 | session=xxx; smac=xxx; |

多账号支持：  
使用 **换行符** 或 **& 符号** 分隔多个 Cookie。

---

### 2. 签到模式（可选）

| 变量名 | 说明 | 默认值 |
|------|------|------|
| DEEPFLOOD_SIGN_TYPE | 签到类型：fixed 或 random | fixed |

fixed：固定签到（默认，奖励稳定）  
random：随机签到（奖励有波动，可能获得更多惊喜）

---

### 3. 高级配置（可选）

如果遇到 **签名错误** 或 **403 错误**，可尝试添加抓包获取的 Headers：

| 变量名 | 说明 | 示例 |
|------|------|------|
| DEEPFLOOD_HEADERS | JSON 格式 Headers | {"refract-key":"...", "refract-sign":"..."} |

---
