# NodeSeek 自动签到脚本

## 📋 脚本简介

这是一个用于 **NodeSeek 论坛** 的自动签到脚本。  
支持 **随机鸡腿签到（默认）** 和 **固定签到**。

---

## ⚙️ 配置说明

### 1. 环境变量（必需）

在 **青龙面板** 中添加环境变量：

| 变量名 | 说明 | 示例 |
|------|------|------|
| NODESEEK_COOKIE | 用户的 Cookie 字符串 | 此处填写你的完整 Cookie |

多账号支持：  
如果有多个账号，请在 NODESEEK_COOKIE 中使用 **换行符** 或 **& 符号** 分隔不同的 Cookie 字符串。

---

### 2. 签到模式配置（可选）

| 变量名 | 说明 | 默认值 |
|------|------|------|
| NODESEEK_SIGN_TYPE | 签到类型：random 或 fixed | random |

random：随机签到（有机会获得更多鸡腿，推荐）  
fixed：固定签到

---

### 3. 获取 Cookie 方法

1. 浏览器访问 https://www.nodeseek.com 并登录  
2. 按 F12 打开开发者工具  
3. 切换到 Network（网络） 标签  
4. 刷新页面或点击任意链接  
5. 找到一个主域名下的请求（如 board 或 www.nodeseek.com）  
6. 复制 Request Headers 中的 Cookie 值  

---

## 📦 依赖

脚本依赖 axios：

npm install axios
